# Numerical Analysis

Implementation of numerical methods for a Numerical Analysis course in Computer Science.

## Implemented Methods

### Root Finding
- Bisection Method
- Newton-Raphson Method
- Secant Method

### Linear Systems
- Gaussian Elimination
- LU Decomposition
- Jacobi Method
- Gauss-Seidel Method

### Interpolation
- Lagrange Interpolation
- Newton Interpolation

### Numerical Integration
- Trapezoidal Rule
- Simpson's Rule

### Ordinary Differential Equations
- Euler Method
- Runge-Kutta Method

## Goal

This repository contains implementations created to understand numerical algorithms and their computational behavior.
